geojson.io

GeoJSON is a simple, popular format for map data. geojson.io lets you edit it.

This is an open source project that you can contribute to and improve.

Getting Started

Try an example GeoJSON file with a few basic elements. You can click on the buttons on the left to add and remove geometry, and edit the code on the right. If you make a mistake, a pink box will appear and give hints on mouseover.

Protips

Hit Cmd-S (or Ctrl-S) to 'save' what you're working on.
